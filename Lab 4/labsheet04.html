CITS1401 Problem Solving and Programming

Labsheet 4

Simple Graphics Programming

This labsheet is in two parts: a number of review exercises taken from the textbook that reinforce concepts from the lectures; and an important problem-solving technique, motivated by its use in solving a larger-sized problem representative of the types of problems that can be solved with computers.

Review Exercises (Zelle, Chapter 3 & Chapter 4 upto 4.3)

  1. Write a program that calculates the total cost of material required to make a sphere of certain radius. Prompt the user for radius in meters and material price per square meter.  Area of sphere = 4*pi*r*r

  2. Write a program that prompts the user for the radius of a circle and then draws it using the graphics.py library. Hint: Put graphics.py in the same directory as your code and import it.

  3. Calculate the circumference and area of the circle and print it in the graphics window under the circle. Hint: use graphics.Text(graphics.Point(x,y), 'area').draw(win) where win is the name of your graphics window object.

  4. Write a program that will draw a rectangle on a graphics window and move it horizontally.  Hints: use the graphics.Rectangle object and  the move method. Use time.sleep() to give a delay.

  5. Try increasing and decreasing the speed of the rectangle.

  6. Write a program to draw 20 concentric circles using a loop where the radius doubles each time. You will need to be careful of the size of the graphics window.

  7. Write a program where user is asked for the option of drawing square or circle or exit. On the selection of the option, program asks for details of square or circle and draws them. Afterwards, program repeatedly asks for the options till exit is selected or user inputs options 10 times.  Use different colors for each drawing for recognition purpose. 

  8. (Challenge) Write a program to draw a triagle which takes three points as inputs. 

  9. (More Challenging) Write a program which asks user for a number and draw the number of equal size squares on the window. Make the colors different to recognize different squares. If number is not valid then terminate the program with a message. For example, if user inputs 4, then 4 squares (50 x 50 pixels) are to be drawn in the window (200 x 200 pixels) such that there are in 2 rows and 2 columns. Similarly, if user inputs 7 the program is terminated with message. Hint: find square root of the number.

  10. (Challenge) Write a program which creates a button (make a rectangle and assigned different color). When button is clicked, it disappears and appears randomly at any other place in the window. The process is repeated 10 times. Remember, button should only disappear when area of button is clicked.
